# PWA Icons

Para o PWA funcionar completamente, você precisa adicionar os seguintes arquivos nesta pasta:

## Arquivos necessários:

- `logo-192.png` - Ícone 192x192px para PWA (purpose: any maskable)
- `logo-512.png` - Ícone 512x512px para PWA (purpose: any maskable)
- `favicon.ico` - Favicon para o navegador

## Como criar os ícones:

1. Crie um logo quadrado (ex: 512x512px)
2. Use ferramentas online como:
   - https://realfavicongenerator.net/
   - https://favicon.io/
   - https://www.websiteplanet.com/webtools/favicon-generator/

3. Ou use ferramentas de design como Figma/Photoshop para exportar nas dimensões corretas

## Especificações:

- **192x192px**: Usado em telas de splash e home screen em dispositivos Android
- **512x512px**: Usado em PWA install prompts e telas de alta resolução
- **maskable**: Permite que o ícone seja adaptado para diferentes formas (círculo, quadrado arredondado)
- **Background**: Use fundo sólido ou transparente dependendo do design

## Dica:

O `logo.svg` atual é um placeholder. Substitua os arquivos PNG para personalizar sua PWA!
